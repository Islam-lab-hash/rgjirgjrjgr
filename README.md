# Telegram магазин

Бот-магазин на `aiogram 3` с редактируемым каталогом, корзиной, заказами и админ-панелью.

## Запуск

```bash
pip install -r requirements.txt
python bot.py
```

Настройки берутся из `.env`:

```env
BOT_TOKEN=...
ADMIN_IDS=123456789
```

## Команды

| Команда | Назначение |
| --- | --- |
| `/start` | Главное меню |
| `/myid` | Показать Telegram ID |
| `/admin` | Админ-панель |

## Что можно менять в админке

- Товары: добавление, удаление, редактирование названия, цены, размеров, цветов, материала, страны, описания, фото, категории и статуса показа.
- Категории: добавление, переименование и удаление.
- Тексты: стартовый текст, адреса, доставка, оплата, обмен, Instagram, оператор.
- Картинки: старт, каталог, подбор, корзина, информация.
- Меню: названия кнопок главного меню.
- Заказы: просмотр, подтверждение и отмена.
- Админы: добавление и удаление дополнительных админов.
- Рассылка: отправка текста всем сохраненным пользователям.

## Изображения разделов

Стандартные картинки лежат в `assets`:

- `start.png`
- `catalog.png`
- `selector.png`
- `cart.png`
- `info.png`

Если в админке загрузить новую картинку для раздела, бот будет использовать Telegram `file_id`. Если отправить `-`, бот снова возьмет стандартную картинку из `assets`.

## Импорт товаров из ZIP

Фото из архивов с рабочего стола импортированы как товары:

- Бюсты: 6 товаров
- Комплекты: 3 товара
- Пеньюары: 1 товар
- Пижамы: 6 товаров
- Халаты: 7 товаров

Для повторного импорта можно запустить:

```bash
python seed_zip_products.py
```

Скрипт сначала очищает товары и корзины, затем заново добавляет товары из уже извлеченных фото в `assets/products`.

## База

SQLite база создается автоматически в `store.db`.
